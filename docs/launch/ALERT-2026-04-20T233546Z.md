3. OpenSanctions rate-limit cascade — if the sanctions featured path is being called inside the signals response assembly, the 60 req/hour cap can bubble up as a 500. (See `project_opensanctions_rate_limits.md`.)
4. Schema drift regression on `alert_tier` / enums / `source_ids` — previously resolved Apr 15 but worth re-verifying. (See `project_signals_schema_drift.md`.)
